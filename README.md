# PNJU-TodayWidget

+ A Mac OS X Yosemite today widget, making it easier to connect to `p.nju.edu.cn`.

## How To Use

1. Download and open the project.
2. Change your `username` and `password` in `TodayViewController.m`.
3. Build and run.
4. Enjoy!
